Personaliza respuestas basadas en parámetros, queries o cuerpo de la petición.

Ejemplos de configuración

Ejemplo simple GET

{
  "method": "GET",
  "path": "/api/users",
  "responses": [
    {
      "status": 200,
      "body": [
        { "id": 1, "name": "Ana" },
        { "id": 2, "name": "Luis" }
      ]
    }
  ]
}

Ejemplo con condiciones

{
  "method": "GET",
  "path": "/api/users/:id",
  "responses": [
    {
      "where": {
        "params.id": "1"
      },
      "status": 200,
      "body": { "id": 1, "name": "Ana" }
    },
    {
      "where": {
        "params.id": "2"
      },
      "status": 200,
      "body": { "id": 2, "name": "Luis" }
    },
    {
      "status": 404,
      "body": { "error": "Usuario no encontrado" }
    }
  ]
}
